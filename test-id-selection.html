Method 2: ID-Based Selection (NEW)

These buttons are attached via data-mysellkit-buttons="buy-btn-1,buy-btn-2" on the script tag:

<script src="checkout.js" data-mysellkit-buttons="buy-btn-1,buy-btn-2"></script> <button id="buy-btn-1" data-product-id="XXX">Buy Template 1</button> <button id="buy-btn-2" data-product-id="YYY">Buy Template 2</button>

Method 1: Data-Attribute Selection (EXISTING)

These buttons use the original data-mysellkit-checkout attribute:

<button data-mysellkit-checkout="XXX">Buy Template</button>
